## problexity v0.3.0 {{  date: '2022-06-19T00:00Z' }}

Publication of the first public version of the problexity module. It is an open-source python library containing the implementation of measures describing the complexity of the classification and regression problems. the package contains a complexitycalculator model, allowing the calculation, analysis and visualization of problem complexity measures.
